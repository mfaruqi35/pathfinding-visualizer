# Pathfinding Visualizer 🧩

A simple visualizer for **BFS** and **DFS** algorithms solving a maze. Watch how each algorithm finds its way from start to goal, step by step.

---

## 🗺️ How it works

The maze is loaded from a `.txt` file, and the program will visualize how BFS and DFS explore the maze to find a path from the start point to the goal.

---

## 🧰 Tech Stack

- **Python**
- **Pygame**

---

## 🚀 Getting Started

**1. Clone the repo**

```bash
git clone https://github.com/mfaruqi35/pathfinding-visualizer.git
cd pathfinding-visualizer
```

**2. Install dependencies**

```bash
pip install pygame
```

**3. Run**

```bash
python main.py
```

---

## 📄 Maze Input Format

The maze is read from a `.txt` file. Make sure your maze file is formatted correctly before running the program.
